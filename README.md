# Clothes-shop

#Web tạo bởi IDE Eclipse

#Wev sử dụng Java Spring boot, Jsp, Css, Bootstrap và Microsoft Sql Server.

#Web có các chức năng:
  - Phân trang.
  - Lọc sản phẩm.
  - Giỏ hàng.
  - Chi tiết sản phẩm.
  - Đăng nhập, đăng kí, đăng xuất.
  - Quản lý người dùng.
  - Quản lý danh mục.
  - Quản lý sản phẩm.
